Création de bases de données

C'est un projet réalisé en binôme qui consistait un créer un base de donnée à partir d'un cahier des charges client. Réussir un certains nombre de requêtes sur cette base et enfin proposer un jeu de données test pour peupler la base.

Pour cela, plusieurs étapes de réalisation :

  • Critique de la base de données existante et proposition d'un nouveau modèle et sa transformation en modèle relationnel

  • Répondre aux différentes requêtes

  • Extension de la base de données et proposition d'un jeu de données test pour peubler la nouvelle base de données

Compétences développées :

  • Concevoir et mettre en place une base de données à partir d'un cahier des charges

  • Exploiter/analyser les données via des requêtes SQL

  • Migrer une base de données d'un ancien schéma vers un nouveau

Exploitation de base de données

Comme le projet précédent, ce projet a été réalisé en binôme
Nous devion traduire un modèle de données en un schéma relationnel, peupler la base à partir de données réelles (données parcoursup), concevoir des vues pour exploiter les données et enfin présenter ce projet à l'oral.

Voici les étapes réalisées pour terminer le projet :

  • Traduire un modèle de données en un schéma relationnel

  • Peupler la base à base des données réelles de Parcoursup

  • Concevoir les vues pour exploiter les données et en faire des synthèses

  • Présentation orale

Compétences développées :

  • Concevoir et mettre en place une base données à partir d'un cahier des charges client

  • Visualiser des données
